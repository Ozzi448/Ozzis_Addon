**Ozzis Installer v1.0.0.2**
  * Changed to different code, resorted to use python, there are more pros than cons compared to the old script.
  * Added File > Preferences > Dropdown for Client Types.
  * Added Theme control; Changeable between (System, Dark, and Light mode).
  * Added the ability to toggle Glass with transparency on or off; *default* = off.
  * Added the ability to click a button to make a shortcut on the desktop, if one exists it will popup with error.
  * Added a check box option to load when Windows starts up.
  * Added the ability for the application to remember the last position it was moved to on the screen before it was closed.
    * Made the necessary changes to open the Preferences and Changelogs to open where the Main Frames are positioned.
  * Added a new Reinstall button, had to change the size of the Action Button.
  * Changed the Action Button to show red when not installed or out of date, Green when up to date.
  * Fixed a bug with Light mode, if selected, Preferences window would disappear before you get a chance to save adn close thus making the application not respond forcing End Tasking.
  * Fixed Glass / Transparency bug.
  * Added Markdown compatibility. This is beta as it is still being worked on.
  * Changed the settings file, I am now using .dat file and this is now encrypted to prevent any unauthorised tampering, more secure now.
  * Fixed a new bug where the application loads with no settings file that presented with a small frame.
